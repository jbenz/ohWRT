# ohWRT
OpenWRT Scripts

# Getting Started
> [!NOTE]
> The easiest way to get started by installing dependencies and setting file permissions, is to ssh as root to your router and run:
```
git clone http://jbenz/ohWRT
/bin/ash ~/ohWRT/setup.sh
```
